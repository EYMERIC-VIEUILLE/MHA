Voici une liste des commandes de MHA :
--------------------------------------

startserver : lance un serveur
join xxx : se connecte au serveur de la machine xxx
execute xxxx : permet d'executer les commandes du fichier xxxx

Création de perso
-----------------

addtroll id nom race : créé un troll de la race "race" : 
      0 = skrim
      1 = durakuir
      2 = kastar
      3 = tomawak
      4 = darkling
carac Niv Att Esq Deg Reg PV Vue Durée_DLA(en minutes) MM_pure RM_pure : donne les caracs de son troll
addcomp num pourcentage : indique que l'on a la comp n° num à pourcentage %
      1  = BS
      2  = RA
      3  = AM
      4  = Camou
      5  = Balayage
      6  = AP
      7  = Charger
      8  = piège
      9  = contre attaque
      10  = CdB
      11 = DE
      12 = Frénésie
      13 = LdP (non fonctionnel)
      14 = Parer
      15 = Pistage (je sais pas s'il marche :) )
addsort num pourcentage : indique que l'on a le sort n° num à pourcentage %
      1  = Hypno
      2  = RP
      3  = Vampirisme
      4  = Projo
      5  = Siphon des âmes
      6  = AA
      7  = AE
      8  = AdA
      9  = AdE
      10  = ADD
      11  = BaM
      12 = BuM
      13 = Explo
      14 = FP
      15 = FA
      16 = Glue
      17 = GdS
      18 = Invisibilité
      19 = Projection
      20 = Sacro
      21 = TP
      22 = VA
      23 = VL
      24 = VLC
      25 = VT
addequip id type Att Esq Deg DLA reg ArN pv vue armure_physique armure_magique effet_de_zone bidouille poids(en minute) mm rm est_équipé nom
     Ajoute un équipement. est_équipé vaut 1 si l'équipement est équipé. effet_de_zone vaut 1 pour les potions à effet de zone et bidouille ne sert à rien pour l'instant (à mettre à 0)  Les différentes valeurs de type

validtroll : indique que l'on a finit de créer son troll

Lancer le jeu
-------------

startgame

Pendant le jeu
--------------

* Toujours disponible :

getprofil : infos sur son troll
getinfostroll xxx : le profil public du troll n° xxx
exit : quitter la partie

* Lorsque que c'est votre tour de jeu (vous venez de recevoir un message begin XXX) :

begindla : on veut jouer de suite sa DLA et l'activer
decaletour xxx : ne pas jouer de suite mais dans xxx minutes
getvue : obtenir les trolls sa vue 
getlieux : obtenir les lieux dans sa vue

* Pendant la partie

getvue : comme précédemment
getlieux : obtenir les lieux dans sa vue
getinfoslieu : obtenir des infos sur le lieu sur lequel on se trouve
releve : releve le troll
deplace x y n : se déplacer de x cases en X, y cases en y, n cases en N. Un déplacement normal quoi
attaque xxx : permet d'attaquer normalement le troll xxx (ne marche que s'il l'on est sur sa case, 
    et qu'il est visible
comp xxx [pleins de paramètres] : essaie de lancer la comp xxx (voir les numéros de comp si dessus).
    Suivant la comp que vous voulez lancer, il y aura besoin de mettre des paramètres (le numéro d'un
    troll, les déplacements souhaités...)
sort xxx [pleins de paramètres] : essaie de lancer le sort xxx (voir les numéros de sort si dessus).
    Suivant la sort que vous voulez lancer, il y aura besoin de mettre des paramètres (le numéro d'un
    troll, les déplacements souhaités, la valuer de PV sacrifiés...)
enddla : vous avez fini de jouer votre DLA
decaletour xxx : demander de finir de jouer sa DLA dans xxx minutes
getpa : connaitre le nombre de PA qu'il te reste
